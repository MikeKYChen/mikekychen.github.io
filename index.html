Pokemon 對戰遊戲 （全端專案）

「融合經典與創新的網頁版寶可夢對戰系統」

本作品是一款純網頁開發的互動式寶可夢遊戲，核心設計深度致敬早期 Gameboy 的經典懷舊風格，並完美融合現代 Pokémon Ga-Olé 機台的戰鬥邏輯與震撼動畫。在復刻完整戰鬥系統的基礎上，我更注入了獨特的自創機制與視覺巧思，將兒時的夢想轉化為具體的遊戲體驗，實現了對自製遊戲的極致追求。

※ 本專案透過 PokeAPI 取得寶可夢資料，玩家資料儲存於瀏覽器 LocalStorage，可直接線上遊玩。

TypeScript Vue Vite PokeAPI LocalStorage Animation UI/UX

QR Code 點餐系統 （全端專案）

全方位餐廳數位化解決方案，整合顧客點餐、廚房管理、收銀結帳與數據分析四大核心功能。

採用現代化 UI 設計（漸層背景、毛玻璃效果、流暢動畫），提供直覺的操作體驗。支援 QR Code 掃碼點餐、即時訂單更新、多維度數據統計與圖表分析。整合拖曳式菜單管理、圖片上傳（Imgur）與 JWT 身份驗證，完整實現餐廳經營所需的前後台功能。

※ 本專案需搭配後端伺服器（Express + MongoDB）運行，無法直接透過靜態網頁瀏覽。

Vue 3 TypeScript Express MongoDB JWT Chart.js QR Code RWD

吃啥喝啥 - 美食決策轉盤 （前端專案）

「當你不知道下一餐吃什麼時，就讓命運的轉盤為你決定！」

一款具備互動式轉盤動畫的美食決策工具，支援「吃啥」和「喝啥」雙模式切換。整合菜系標籤分類（10 種菜系）、價格標記、權重調整、收藏與黑名單功能，更提供 5 種心情模式（普通、心情低落、慶祝、趕時間、約會）自動調整推薦邏輯。介面採用玻璃擬態設計搭配流暢動畫，支援歷史記錄、JSON 匯出匯入與完整響應式布局。

※ 本專案原設計整合 Google Places API 自動搜尋附近餐廳，但因未申請 API Key，該功能無法使用。目前可透過手動新增餐廳與本地儲存進行完整體驗。

Vue 3 TypeScript Vite Canvas API LocalStorage Animation RWD UI/UX
