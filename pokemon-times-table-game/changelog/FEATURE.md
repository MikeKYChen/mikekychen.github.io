# 未來項目

### **班級座位表管理系統**
- 編輯模式：拖曳式座位編輯器，支援學生分配和座位調整
- 瀏覽模式：即時查看班級座位、學生成績與統計
- 排名徽章：自動顯示班級前三名學生（🥇🥈🥉）
- 網格吸附：80px 網格系統，拖曳時自動對齊
- 雙模式拖曳：一般拖曳分配學生，Shift+拖曳調整位置


### 優化改進
- 🔧 班級成員載入改用專用 API，不再限制前 50 名
- 💾 座位配置自動儲存至 Google Sheets
- 📊 統計數據排除老師座位計算
- 🎨 操作說明可關閉並重新開啟
